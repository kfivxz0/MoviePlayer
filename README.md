# 비디오 재생 앱 만들기
IOS에서 재공하는 AVPlayerViewController를 사용하면 앱 내부에 저장되어 있는 비디오 파일뿐만 아니라 외부에 링크된 비디오 파일도 간단하게 재생할 수 있습니다. 비디오 재생 방법을 잘 활용하면 쉽게 다음과 같은 비디오 재생 앱도 만들 수 있습니다.
